# Human Resource Management System (HRMS)

## Overview

A full-stack Human Resource Management System developed using Java, JDBC, MySQL, HTML, CSS, and JavaScript. The application provides employee management, department management, attendance tracking, leave management, dashboard analytics, and role-based access control.

## Features

### Authentication

* Secure Login System
* Role-Based Access Control (Admin / Employee)

### Employee Management

* Add Employee
* Update Employee
* Delete Employee
* Search Employee

### Department Management

* Create Departments
* Manage Department Information

### Attendance Management

* Check-In / Check-Out
* Attendance Tracking
* Attendance Dashboard

### Leave Management

* Apply Leave
* Approve / Reject Leave
* Leave Status Tracking

### Dashboard

* Employee Statistics
* Department Statistics
* Attendance Analytics
* Leave Analytics

## Technologies Used

### Backend

* Java
* JDBC
* MySQL

### Frontend

* HTML
* CSS
* JavaScript

### Database

* MySQL

## Roles

### Admin

* Manage Employees
* Manage Departments
* Manage Attendance
* Approve/Reject Leave Requests
* View Dashboard Analytics

### Employee

* View Attendance
* Apply Leave
* View Personal Dashboard
